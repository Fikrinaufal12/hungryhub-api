# 🍽️ HungryHub - Restaurant Menu Management API

REST API untuk mengelola restoran dan menu makanan/minuman.  
Dibangun menggunakan Node.js, Express, Sequelize, dan MySQL.

---

## 🚀 Live API

👉 https://YOUR-RENDER-URL.onrender.com

---

## 🛠 Tech Stack

- Node.js
- Express.js
- Sequelize ORM
- MySQL (Railway)
- Thunder Client / Postman
